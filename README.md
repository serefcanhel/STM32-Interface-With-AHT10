# STM32-Interface-With-AHT10
  
  => In this project, the AHT10 Temperature and Humidity sensor was implemented to the STM32F407VG.
  
  
# Connections
  => AHT10 Temperature and Humidity sensor uses I2C serial communication.
  
  =>      AHT10         STM32F407VG
          -----         ------------
           VIN    ==>       3V
         Ground   ==>     Ground
           SCL    ==>      PB6
           SDA    ==>      PB7
